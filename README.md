Plan Pokrivenosti Nastave – Spring Boot & React Aplikacija

Aplikacija Plan pokrivenosti nastave omogućava kreiranje i pregled plana nastave po predmetima, oblicima nastave i školskim godinama. Izgrađena je kao Full-Stack projekat sa:

Backend: Spring Boot (Java)

Frontend: React.js

Baza: MySQL

Funkcionalnosti
Registracija i prijava

Kreiranje korisničkog naloga

Prijava putem email + lozinka

 Uloga: ADMIN

Administratori mogu:

✔ Kreirati plan pokrivenosti nastave

✔ Videti listu svih planova

Uloga: KORISNIK

Korisnici mogu:

✔ Samo pregledati plan pokrivenosti nastave

Nema uređivanja, brisanja ili dodavanja entiteta – fokus je isključivo na prikazu i kreiranju plana (admin).
